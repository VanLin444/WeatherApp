# WeatherApp
Веб-приложение для получения актуальной информации о погоде с использованием OpenWeatherMap API.

## О проекте
Проект разработан для практики работы с:
- PHP Backend
- REST API
- Асинхронными запросами
- Геолокацией браузера
- Кэшированием данных
- LocalStorage

При загрузке страницы приложение пытается определить местоположение пользователя и автоматически
отображает погоду для текущих координат. Если доступ к геолокации запрещен,
загружается город по умолчанию.

## Функциональность
### Поиск городов
- Автодополнение при вводе
- Поиск через API
- Выбор города из списка

### Геолокация
- Определение координат пользователя
- Автоматическая загрузка погоды

### История поиска
- Хранение последних 5 городов
- LocalStorage

### Кэширование
Сервер сохраняет результаты запросов для уменьшения количества обращений к внешнему API и
ускорения загрузки.
